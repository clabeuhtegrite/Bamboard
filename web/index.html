Bamboard

Flash the latest firmware to your device — straight from this page.

⚠️ This browser can't talk to USB serial devices. Open this page in Chrome, Edge or Opera on a desktop to flash. ⚠️ Web Serial needs a secure (https) page.

Installs Bamboard latest · ESP32-S3

How to flash

  1. Use Chrome, Edge or Opera on a computer (Web Serial isn't available in Firefox, Safari or on phones).
  2. Plug the board into a USB-C data cable (not charge-only).
  3. Click Connect above and pick the serial port that appears.
  4. If no port shows up or the connection fails, put the board in download mode: hold BOOT, tap RST, release BOOT, then retry.
  5. Wait for the flash to finish, then follow the on-screen Wi-Fi + Bambuddy setup (the device opens a Bamboard-setup Wi-Fi access point on first boot).

You only need to do this once. After the first flash, the device updates itself over the air from GitHub Releases at every boot.

Other browsers / manual flashing

Web Serial only works in Chromium-based desktop browsers. On Firefox, Safari, Linux without a Chromium browser, or to flash by hand, download bamboard-factory.bin from the latest release and flash it at offset 0x0 with esptool:

esptool.py --chip esp32s3 write_flash 0x0 bamboard-factory.bin
